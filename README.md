# Contact Manager (Django)

A simple **contact management system** built with Django.  
Users can register, log in, and manage their personal contacts (CRUD).  
Includes pagination, search, categories, profile updates, and image upload.

---

## Features

- User authentication (register, login, logout, update profile, change password).  
- CRUD operations for contacts:  
  - Create, update, delete, and view contact details.  
  - Profile picture upload.  
- Search contacts by name, phone, or email.  
- Categories for contacts.  
- Pagination for large contact lists.  
- Validation rules (e.g., first and last name cannot be identical, unique emails).  

---

## Tech Stack

- Python
- Django
- SQLite (default database)  
- HTML + CSS (base templates)
